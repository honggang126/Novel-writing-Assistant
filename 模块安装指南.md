# 小说写作软件 - 模块安装指南

## 📦 必需安装的模块

### 1. PyQt5 - GUI界面框架
**作用**：创建软件的用户界面
**安装命令**：
```bash
pip install PyQt5
```

### 2. requests - HTTP请求库
**作用**：与AI模型API进行通信
**安装命令**：
```bash
pip install requests
```

### 3. json5 - JSON增强处理
**作用**：支持更灵活的JSON格式解析
**安装命令**：
```bash
pip install json5
```
