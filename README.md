# 小说写作软件

一个基于Python和PyQt5开发的智能小说写作助手，支持多种AI模型集成，帮助用户快速生成小说大纲、章节内容，并提供润色和文件管理功能。

## 功能特性

### 🎯 核心功能
- **智能小说创作**：自动生成小说大纲和章节内容
- **多AI模型支持**：集成Ollama、SiliconFlow、ModelScope等多种AI模型
- **章节润色**：对生成的章节内容进行优化和润色
- **文件管理**：自动保存章节到指定目录，支持规范命名

### 🤖 AI模型支持
- **Ollama**：本地部署的AI模型
- **SiliconFlow**：云端AI服务
- **ModelScope**：阿里云AI模型平台
- **流式响应**：实时显示AI生成内容
